# 통합 LMS 크롤러

여러 대학의 LMS 학습 데이터를 한 곳에서 확인할 수 있는 데스크탑 앱입니다.

## 지원 대학
- 대구대학교 (Hello LMS)
- 경북대학교 (Canvas LMS)

## 사용 방법

1. 이 프로젝트를 다운로드(또는 `git clone`)합니다.
2. 폴더 최상단의 **`LMS Crawler.exe`** 파일을 더블클릭하여 실행합니다.
3. 소속 학교를 선택하고 학번/비밀번호를 입력하여 로그인합니다.
4. 자동으로 수강 과목 및 학습 데이터를 불러옵니다.

> ⚠️ Windows SmartScreen 경고가 나타날 수 있습니다. "추가 정보" → "실행"을 클릭하세요.

## 개발자용

```bash
# 의존성 설치
npm install

# 개발 실행
npm start

# 빌드 (.exe 생성)
npm run build
```
